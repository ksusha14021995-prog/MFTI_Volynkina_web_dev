## Микросервисы

| Сервис | Ответственность | БД |
|--------|-----------------|----|
| `catalog-service` | Товары, бренды, страны, варианты объёма, остатки | `catalog_db` |
| `orders-service` | Корзина, заказы, позиции, статусы, точки самовывоза, контакты покупателей | `orders_db` |
| `admin-service` | Авторизация и сессии администраторов | `admin_db` |

Перед сервисами — API Gateway. Подробности — в [02-architecture.md](02-architecture.md) и [08-microservices.md](08-microservices.md).
