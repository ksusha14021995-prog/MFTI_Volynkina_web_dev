# Документация

Техническое задание на разработку интернет-магазина завода лампочек.

## Структура

ТЗ собирается из 6 артефактов, каждый в своём файле:

| # | Артефакт | Файл |
|---|----------|------|
| 1 | Верхнеуровневая архитектура | `01-architecture.md` |
| 2 | Прототипы интерфейсов | `02-prototypes.md` |
| 3 | Структура БД (по сервисам) | `03-database.md` |
| 4 | Документация API (Postman) | `04-api/` |
| 5 | Описание логики (UML) | `05-logic.md` |
| 6 | Архитектура микросервисов | `06-microservices.md` |

Файлы создаются по ходу выполнения ДЗ 1.
