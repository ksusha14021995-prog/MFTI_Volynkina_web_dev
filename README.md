# MFTI_Volynkina_web_dev

Проект интернет-магазина парфюмерии на разлив «КОМНАТА 26» в рамках 2 семестра магистратуры МФТИ × Нетология, дисциплина «Веб-разработка».

| Папка | Что внутри |
|-------|-----------|
| [`docs/`](docs/) | Техническое задание: продуктовые решения, архитектура, ER-диаграмма, Postman-коллекции, прототипы. ДЗ 1. |
| [`backend/`](backend/) | Микросервисы catalog + orders на FastAPI + PostgreSQL. ДЗ 2. |

Прототипы развёрнуты на GitHub Pages: https://ksusha14021995-prog.github.io/MFTI_Volynkina_web_dev/04-prototypes/
