# MFTI Volynkina — Web Development

Учебный проект магистратуры МФТИ × Нетология «Разработка IT-продукта», 2 семестр — **Веб-разработка** (BHEMBD-25-VR-2).

В рамках курса последовательно проектируется и реализуется веб-приложение микросервисной архитектуры.

## Продукт

**Интернет-магазин завода лампочек** — каталог из 20 наименований, корзина, оформление заказов, админка для управления товарами и заказами.

## Структура репозитория

| Папка | Содержимое |
|-------|------------|
| [`docs/`](docs/) | Техническое задание: архитектура, прототипы, БД, API, UML-диаграммы |
| [`backend/`](backend/) | Backend-микросервисы (товары, заказы) |
| [`frontend/`](frontend/) | Frontend для покупателей |
| [`admin/`](admin/) | Frontend и backend админ-панели |
